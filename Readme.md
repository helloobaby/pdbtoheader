## A IDA plugin

基于IDA,将所有符号转成偏移存储在头文件里。

Download release :  https://anonfiles.com/FcpfabZcxf/pdbtoheader64_dll

用法:  

拷贝dll到E:\IDA Pro 7.6\plugins目录
